## 🎮 게임 소개

클래식 아케이드 게임 갤러가(Galaga)를 현대적으로 재해석한 2D 슈팅 게임입니다.
8종의 다양한 무기, 핵폭탄 궁극기, 콤보 시스템 등 풍부한 게임플레이를 제공합니다.

### 스크린샷

```
[게임 플레이 화면]
- 플레이어 우주선
- 다양한 적들
- 파워업 아이템
- UI (점수, 목숨, 레이더 등)
```

---

## ✨ 주요 기능

### 게임 시스템
- ⚔️ **8종 무기 시스템** - Normal, Laser, Missile, Homing, Spread, Railgun, Plasma, Wave
- 💣 **핵폭탄 궁극기** - 화면 전체 적 즉사
- 🎯 **콤보 시스템** - 최대 x5 점수 배율
- ⭐ **6종 파워업** - Speed, Power, Triple Shot, Shield, 1UP, Rapid Fire
- 👾 **6종 적 타입** - 일반, 빠른 적, 탱크, 카미카제, 분열, 보스

### 게임 기능
- 📡 **레이더 시스템** - 실시간 적/파워업 위치 표시
- 💥 **화면 효과** - 흔들림, 섬광, 폭발 애니메이션
- 🎵 **사운드 시스템** - BGM 및 효과음 자동 생성
- 🎚️ **난이도 선택** - Easy / Normal / Hard
- 🏆 **최고 점수 저장** - 로컬 저장

## 📦 설치 방법

### 요구 사항

- Python 3.12 이상
- Pygame 2.6 이상
- NumPy

### 설치 명령어

```bash
# Python 3.12 설치 확인
python --version

# 필요 패키지 설치
pip install pygame numpy --break-system-packages
```

### 게임 파일 다운로드

```bash
# Git clone (저장소가 있는 경우)
git clone https://github.com/yourusername/galaga-clone.git
cd galaga-clone

# 또는 ZIP 파일 다운로드 후 압축 해제
```

## 🔫 무기 시스템

### 무기 목록

| 무기 | 색상 | 데미지 | 발사속도 | 특징 |
|------|------|--------|----------|------|
| **Normal Shot** | 시안 | 1 | 250ms | 기본 무기, 빠른 연사 |
| **Laser Beam** | 빨강 | 2 | 400ms | 적 관통, 중간 데미지 |
| **Missile** | 주황 | 3 | 600ms | 고데미지, 느린 발사 |
| **Homing Shot** | 초록 | 1 | 350ms | 적 자동 추적 |
| **Spread Shot** | 마젠타 | 1 | 300ms | 5방향 동시 발사 |
| **Railgun** | 시안 | 5 | 800ms | 초고속 관통, 최강 화력 |
| **Plasma Cannon** | 보라 | 4 | 500ms | 폭발형 고데미지 |
| **Wave Beam** | 노랑 | 2 | 200ms | 파동 패턴, 빠른 연사 |

### 무기 추천

- **일반 적**: Normal Shot, Spread Shot
- **보스전**: Railgun, Missile
- **다수 적**: Laser Beam, Wave Beam
- **회피 위주**: Homing Shot

## 💣 궁극기

### 핵폭탄 (Nuclear Bomb)

- **발동 키**: X
- **충전 방식**: 적 처치당 5%씩 충전
- **효과**: 화면 전체 범위의 모든 적 즉사
- **지속 시간**: 1초
- **시각 효과**: 폭발 파동 + 화면 흔들림 + 섬광

### 게이지 표시

- **위치**: 우측 하단
- **색상**:
  - 0-99%: 노란색
  - 100%: 빨간색 깜빡임
- **텍스트**: 
  - 충전 중: "ULTIMATE XX%"
  - 완충: "[X] NUKE!"

## ⭐ 파워업

### 파워업 종류

| 아이템 | 아이콘 색상 | 효과 | 지속시간 |
|--------|-------------|------|----------|
| **SPEED UP** | 초록 | 이동속도 1.5배 증가 | 10초 |
| **POWER UP** | 빨강 | 공격력 2배 증가 | 15초 |
| **TRIPLE SHOT** | 파랑 | 3방향 동시 발사 | 12초 |
| **SHIELD** | 시안 | 1회 피격 무효화 | 8초 |
| **1UP** | 마젠타 | 목숨 +1 | 영구 |
| **RAPID FIRE** | 노랑 | 연사속도 2배 증가 | 10초 |

### 드롭 확률

- 일반 적: 15% (Normal 난이도 기준)
- 보스: 100% (좋은 아이템 확정)

### 파워업 중첩

- 같은 파워업 획득 시 시간 연장
- 다른 파워업은 동시 사용 가능

## 👾 적 종류

### 적 타입

| 적 | 색상 | HP | 속도 | 점수 | 특징 |
|----|------|-----|------|------|------|
| **일반** | 회색 | 1 | 2.0 | 100 | 기본 적 |
| **빠른 적** | 초록 | 1 | 5.0 | 150 | 빠른 이동, 낮은 발사율 |
| **탱크** | 파랑 | 4 | 1.2 | 200 | 높은 체력, 느림 |
| **카미카제** | 주황 | 1 | 2.4 | 130 | 플레이어 추적 돌진 |
| **분열** | 보라 | 2 | 1.6 | 150 | 죽으면 2개로 분열 |
| **보스** | 빨강 | 5 | 2.0 | 500 | 트랙터 빔 발사 |

### 등장 시기

- **웨이브 1-2**: 일반 적만
- **웨이브 3+**: 빠른 적 등장
- **웨이브 5+**: 탱크 등장
- **웨이브 7+**: 카미카제 등장
- **웨이브 9+**: 분열 적 등장
- **보스**: 랜덤 등장

## 🎯 콤보 시스템

### 콤보 조건

- 2초 내에 연속으로 적 처치
- 시간 초과 시 콤보 리셋

### 콤보 배율

| 콤보 수 | 배율 | 색상 |
|---------|------|------|
| 1 | x1.0 | - |
| 2-5 | x1.5 | 흰색 |
| 6-10 | x2.0 | 노랑 |
| 11-20 | x3.0 | 주황 |
| 21-50 | x4.0 | 빨강 |
| 51+ | x5.0 | 마젠타 |

### 점수 계산

```
최종 점수 = 기본 점수 × 콤보 배율 × 난이도 배율
```

예시:
- 일반 적 처치 (100점)
- 10 콤보 (x2.0)
- Normal 난이도 (x1.0)
- 최종: 100 × 2.0 × 1.0 = 200점

## 🛠️ 실행 파일 만들기

### PyInstaller 설치

```bash
py -3.12 -m pip install pyinstaller --break-system-packages
```

### 기존 빌드 삭제 (선택사항)

```bash
cd C:\Users\admin\galaga_clone
rmdir /s /q build
rmdir /s /q dist
del GalagaClone.spec
```

### 실행 파일 생성

```bash
cd C:\Users\admin\galaga_clone
py -3.12 -m PyInstaller --onefile --windowed --name "GalagaClone" --collect-all jaraco --collect-all pkg_resources main.py
```

### 옵션 설명

- `--onefile`: 단일 exe 파일 생성
- `--windowed`: 콘솔 창 숨김
- `--name "GalagaClone"`: 실행 파일 이름
- `--collect-all jaraco`: jaraco 모듈 포함
- `--collect-all pkg_resources`: pkg_resources 포함

### 생성 위치

```
galaga_clone/
├── dist/
│   └── GalagaClone.exe    ← 최종 실행 파일 (배포용)
├── build/                  (삭제 가능)
└── GalagaClone.spec        (삭제 가능)
```

### 배포

- `dist/GalagaClone.exe` 파일만 배포
- Python 설치 불필요
- 파일 크기: 약 30-50MB
- Windows 7/10/11 호환

## 🐛 문제 해결

### 게임이 실행되지 않음

```bash
# 패키지 재설치
pip uninstall pygame numpy
pip install pygame numpy --break-system-packages
```

### 사운드가 나오지 않음

- 시스템 볼륨 확인
- Pygame mixer 초기화 확인

### 실행 파일 오류

```bash
# 콘솔 창으로 빌드 (오류 확인용)
py -3.12 -m PyInstaller --onefile --name "GalagaClone" --collect-all jaraco --collect-all pkg_resources main.py
```

### 프레임 드롭

- 해상도 조정: `settings.py`에서 `SCREEN_WIDTH`, `SCREEN_HEIGHT` 변경
- FPS 조정: `settings.FPS` 값 낮추기
